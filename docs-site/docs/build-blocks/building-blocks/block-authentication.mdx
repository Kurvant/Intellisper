### Block Authentication

Intellisper supports multiple forms of authentication, you can check those forms [here](../block-reference/authentication)

Now, let's establish authentication for this block, which necessitates the inclusion of an API Key in the headers.

Modify `src/index.ts` file to add authentication,

```ts
import { BlockAuth, createBlock } from '@intelblocks/blocks-framework';

export const gelatoAuth = BlockAuth.SecretText({
  displayName: 'API Key',
  required: true,
  description: 'Please use **test-key** as value for API Key',
});

export const gelato = createBlock({
  displayName: 'Gelato',
  logoUrl: 'https://cdn.intellisper.com/blocks/gelato.png',
  auth: gelatoAuth,
  authors: [],
  actions: [],
  triggers: [],
});
```

<Note>
  Use the value **test-key** as the API key when testing actions or triggers for
  Gelato.
</Note>
