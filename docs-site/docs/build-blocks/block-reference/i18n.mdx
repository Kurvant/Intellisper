---
title: 'Block i18n'
description: 'Learn about translating blocks to multiple locales'
---

<Steps>
  <Step title="Generate">
    Run the following command to create a translation file with all the strings that need translation in your block
    ```bash
    npm run cli blocks generate-translation-file BLOCK_FOLDER_NAME
    ```
  </Step>
  <Step title="Translate">
     Make a copy of `packages/blocks/<community_or_custom>/<your_block>/src/i18n/translation.json`, name it `<locale>.json` i.e fr.json and translate the values.
    <Tip>
      For open source blocks, you can use the [Crowdin project](https://crowdin.com/project/activepieces) to translate to different languages. These translations will automatically sync back to your code.
    </Tip>
  </Step>
  
   <Step title="Test Locally">
    After following the steps to [setup your development environment](/build-blocks/building-blocks/development-setup), click the small cog icon next to the logo in your dashboard and change the locale.

     ![Locales](/img/i18n-pieces.png)
    <br></br>
  
    In the builder your block will now appear in the translated language:
    ![French Webhooks](/img/french-webhooks.png)
  </Step>

  <Step title="Publish">
  Follow the docs here to [publish your block](/build-blocks/sharing-blocks/overview)

  </Step>

</Steps>
