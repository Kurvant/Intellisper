GitHub Codespaces is a cloud development platform that enables developers to write, run, and debug code directly in their browsers, seamlessly integrated with GitHub.

### Steps to setup Codespaces

1. Go to [Intellisper repo](https://github.com/Kurvant/Intellisper).

2. Click Code `<>`, then under codespaces click create codespace on main.

![Create Codespace](/img/screenshots/development-setup_codespaces.png)

<Note>
  By default, the development setup only builds specific blocks.Open the file
  `.env.dev` and add comma-separated list of blocks to make
  available.

For more details, check out the [Block Development](/build-blocks/building-blocks/development-setup#blocks-development) section.

</Note>

3. Open the terminal and run `npm start`

4. Access the frontend URL by opening port 4200 and signing in with these details:

Email: `dev@intellisper.local`
Password: `12345678`
