## Methods
- [Contribute Back](/build-blocks/sharing-blocks/contribute): Publish your block by contributing back your block to main repository.
- [Community](/build-blocks/sharing-blocks/community): Publish your block on npm directly and share it with the community.
- [Private](/build-blocks/sharing-blocks/private): Publish your block on Intellisper privately.
