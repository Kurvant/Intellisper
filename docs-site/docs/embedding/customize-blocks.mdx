If you would like to only show specific blocks to your embedding users, we recommend you do the following:

<Steps>
    <Step title="Tag Blocks">
        Tag the blocks you would like to show to your user by going to **Platform Admin -> Setup -> Blocks**, selecting the blocks you would like to tag and hit **Apply Tags**

        ![Bulk Tag](/img/screenshots/tag-pieces.png)
    </Step>
    <Step title="Add Tags to Provision Token">
       You need to specify the tags of blocks in the token, check how to generate token in [provisioning users](./provision-users).

        You should specify the `blocks` claim like this:
        ```json
        {
            /// Other claims
            "blocks": {
            "filterType": "ALLOWED",
            "tags": [ "free" ]
        }
        }
        ```

        Each time the token is used by the embedding SDK, it will sync all blocks with these tags to the token's project. 
        The project will only contain the blocks that contain these tags.
    </Step>
</Steps>
