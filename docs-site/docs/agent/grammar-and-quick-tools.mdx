Alongside the full agent, Intellisper offers **quick tools** — focused, single-purpose helpers that give
you an answer in one shot, without running a full agent task.

## Grammar (proofreading)

The grammar tool proofreads a piece of text. Give it your writing, and it returns:

- the **corrected text**, and
- a list of **edits**, each labelled by type — spelling, grammar, punctuation, or style.

The highlighted ranges of what changed are computed precisely by comparing your original text against the
correction, so the highlights always line up with the actual differences.

A few practical details:

- It handles up to **12,000 characters** per request; longer text is truncated.
- It's a **one-shot** tool — it returns a result directly, without the back-and-forth of a full agent
  run.
- It runs on a fast, lightweight model, so it's quick and inexpensive.

Grammar and other quick tools count against your plan's **quick-tools** allowance (see
[Plans & limits](/agent/plans-and-limits)).
