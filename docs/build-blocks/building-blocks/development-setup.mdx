## Prerequisites

- Node.js v18+
- npm v9+

## Instructions

1. Setup the environment

```bash
node tools/setup-dev.js
```

2. Start the environment

This command will start Intellisper with sqlite3 and in memory queue.

```bash
npm start
```

<Note>
  By default, the development setup only builds specific blocks.Open the file
  `.env.dev` and add comma-separated list of blocks to make
  available.

For more details, check out the [Block Development](/build-blocks/building-blocks/development-setup#blocks-development) section.

</Note>

3. Go to **_localhost:4200_** on your web browser and sign in with these details:

Email: `dev@ap.com`
Password: `12345678`


## Blocks Development

When [`IB_BLOCKS_SYNC_MODE`](https://github.com/activepieces/activepieces/blob/main/.env.dev#L14) is set to `OFFICIAL_AUTO`, all blocks are automatically loaded from the cloud API and synced to the database on first launch. This process may take a few seconds to several minutes depending on your internet connection.

For local development, blocks are loaded from your local `dist` folder instead of the database. To enable this, set the [`IB_DEV_BLOCKS`](https://github.com/activepieces/activepieces/blob/main/.env.dev#L5) environment variable with a comma-separated list of blocks. For example, to develop with `google-sheets` and `cal-com`:

```sh
IB_DEV_BLOCKS=google-sheets,cal-com npm start
```
