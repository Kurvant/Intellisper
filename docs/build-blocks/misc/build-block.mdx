You can use the CLI to build custom blocks for the platform. This process compiles the blocks and exports them as a `.tgz` packed archive.

### How It Works

The CLI scans the `packages/blocks/` directory for the specified block. It checks the **name** in the `package.json` file. If the block is found, it builds and packages it into a `.tgz` archive.

### Usage

To build a block, follow these steps:

1. Ensure you have the CLI installed by cloning the repository.
2. Run the following command:

```bash
npm run build-block
```

You will be prompted to enter the name of the block you want to build. For example:

```bash
? Enter the block folder name : google-drive
```

The CLI will build the block and you will be given the path to the archive. For example:

```bash
Block 'google-drive' built and packed successfully at packages/blocks/community/google-drive/dist
```

You may also build the block non-interactively by passing the block name as an argument.  For example:
```bash
npm run build-block google-drive
```
