<Snippet file="enterprise-feature.mdx" />

This guide assumes you have already created a block and created a private fork of our repository, and you would like to package it as a file and upload it.

<Tip>
Friendly Tip: There is a CLI to easily upload it to your platform. Please check out [Publish Custom Blocks](../misc/publish-block).
</Tip>



<Steps>
    <Step title="Build Block">
        Build the block using the following command. Make sure to replace `${name}` with your block folder name.

        ```bash
        npm run blocks -- build --name=${name}
        ```

        <Info>
        More information about building blocks can be found [here](../misc/build-block).
        </Info>
    </Step>
    <Step title="Upload Tarball">
        Upload the generated tarball inside `dist/packages/blocks/${name}`from Intellisper Platform Admin -> Blocks
    
        ![Manage Blocks](/resources/screenshots/install-piece.png)

    </Step>
</Steps>
